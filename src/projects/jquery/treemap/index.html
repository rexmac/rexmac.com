jQuery Treemap v0.1.0

A (simple) jQuery plugin for generating treemaps.

Download

You can download the source on GitHub. Contribution in the forms of bug reports, feature requests, and code improvements are most welcome!

Usage and Demos

Reading data from list elements of an unordered list

HTML:


<ul id="demo-ul">
  <li value="100">A</li>
  <li value="100">B</li>
  <li value="100">C</li>
  <li value="25">D</li>
  <li value="25">E</li>
  <li value="25">F</li>
  <li value="25">G</li>
</ul>

Javascript:

$('#demo-ul').treemap();

Result:

  • A
  • B
  • C
  • D
  • E
  • F
  • G

Without “heatmap”

HTML:


<ul id="demo-ul-without-heatmap">

Custom color range and legend with custom labels

HTML:


<ul id="demo-ul-with-legend">
  <li value="100">A</li>
  <li value="100">B</li>
  <li value="100">C</li>
  <li value="25">D</li>
  <li value="25">E</li>
  <li value="25">F</li>
  <li value="25">G</li>
</ul>

Javascript:


$('#demo-ul2').treemap({
    colors: ['#93c', '#f00'],
    legend: true,
    legendLabels: ['Cold', 'Hot']
});

Result:

  • A
  • B
  • C
  • D
  • E
  • F
  • G
